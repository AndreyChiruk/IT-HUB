# IT-HUB project

# Проект для повторения изученного в университете

<ul>
    <li>TypeScript — весь код</li>
    <li>OOP — классы, абстракции, наследование, полиморфизм</li>
    <li>MongoDB — используется для хранения событий </li>
    <li>PostgreSQL — используется для хранения пользователей</li>
    <li>Express — API Gateway, middleware</li>
    <li>NestJS — микросервисы, DI, модули</li>
    <li>React — основная реализация приложения</li>
    <li>React Native — мобильное приложение</li>
    <li>Docker Compose — оркестрация</li>
    <li>Nginx — reverse proxy, статика</li>
    <li>GitHub Actions — CI/CD пайплайн</li>
    <li>Mocha + Chai — тестирование</li>
</ul>
